# Image Gallery

A modern and responsive Image Gallery built using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

## Features

- Responsive Grid Layout
- Image Categories
- Lightbox Preview
- Next & Previous Navigation
- Hover Animations
- Image Filters
- Keyboard Navigation
- Mobile Friendly

## Technologies Used

- HTML5
- CSS3
- JavaScript
